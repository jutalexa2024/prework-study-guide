HTML Notes

  • HTML is the standard markup language used to create the structure of web pages. It defines elements like headings, paragraphs, links, images, and other content.
  • HTML uses "tags" to create elements. For example, h1 defines a heading, and p defines a paragraph.

CSS

  • CSS is used to style HTML elements. It controls the layout, colors, fonts, spacing, and overall visual appearance of a webpage.
  • CSS uses selectors to target specific HTML elements. For example, p selects all paragraphs, while .className selects elements with a specific class.

GIT

  • GitHub is a platform for version control using Git. It helps track changes in code over time, allowing multiple people to collaborate on projects.
  • Contributors can propose changes to a project via pull requests, allowing for code review and discussion before merging changes into the main branch.

JavaScript

  • JavaScript is a scripting language that enables interactivity and dynamic behavior on webpages.
  • JavaScript is based on functions and variables. You can declare variables with let, const, or var, and create functions to encapsulate logic.
